# 记账 APP 需求文档

## 项目概述

**项目名称**: 记账本 (Bookkeeper)
**版本**: 1.0.0
**平台**: Android (Flutter 跨平台)
**更新日期**: 2026-05-06

---

## 1. 项目背景

用户需要一个功能完整、界面美观的记账 APP，用于日常收支管理。要求：
- 界风格类似 Claude（紫色/靛蓝主色调、简洁极简）
- 支持高级记账功能（多币种、标签、周期记账、数据导出）
- 预留微信/支付宝账单自动导入接口

---

## 2. 功能需求

### 2.1 核心记账功能

#### 2.1.1 收支记录
- **功能描述**: 记录每笔收入和支出
- **输入项**:
  - 金额（必填）
  - 分类（必填，支持二级分类）
  - 账户（必填）
  - 日期（默认当天）
  - 备注（可选）
  - 标签（可选，支持多选）
  - 图片附件（可选）
- **操作**:
  - 新增记录
  - 编辑记录
  - 删除记录
  - 查看记录详情

#### 2.1.2 分类管理
- **功能描述**: 管理收入和支出分类
- **特性**:
  - 系统内置默认分类（不可删除）
  - 用户自定义分类
  - 支持二级分类
  - 分类图标和颜色自定义
  - 分类排序

#### 2.1.3 账户管理
- **功能描述**: 管理不同支付账户
- **默认账户**:
  - 现金
  - 银行卡
  - 支付宝
  - 微信
- **特性**:
  - 账户余额自动计算
  - 账户图标和颜色自定义
  - 是否计入总资产

### 2.2 高级功能

#### 2.2.1 标签系统
- **功能描述**: 为交易记录添加标签，便于分类和筛选
- **特性**:
  - 创建/编辑/删除标签
  - 为交易记录添加多个标签
  - 按标签筛选交易

#### 2.2.2 多币种支持
- **功能描述**: 支持不同货币的交易记录
- **特性**:
  - 选择交易币种
  - 自动获取汇率
  - 汇率缓存（24小时更新）
  - 统计时自动换算为默认币种

#### 2.2.3 周期记账
- **功能描述**: 自动记录固定周期的收支
- **支持频率**:
  - 每日
  - 每周
  - 每月
  - 每年
- **特性**:
  - 创建/编辑/删除周期规则
  - 启用/禁用规则
  - 自动生成交易记录
  - 执行后通知用户

#### 2.2.4 预算管理
- **功能描述**: 设置和跟踪预算
- **特性**:
  - 月度总预算
  - 分类预算
  - 预算进度显示
  - 超支提醒

#### 2.2.5 统计报表
- **功能描述**: 可视化收支数据
- **图表类型**:
  - 饼图：分类占比
  - 折线图：收支趋势
  - 柱状图：分类排行
- **时间维度**:
  - 本周
  - 本月
  - 本季
  - 本年
  - 自定义范围

#### 2.2.6 数据导出
- **功能描述**: 导出交易数据
- **格式**:
  - CSV
  - Excel (.xlsx)
- **特性**:
  - 选择时间范围
  - 选择导出字段
  - 分享导出文件

### 2.3 自动化功能（预留接口）

#### 2.3.1 账单导入
- **功能描述**: 从微信/支付宝导入账单
- **支持来源**:
  - 微信支付账单 (CSV)
  - 支付宝账单 (CSV)
- **特性**:
  - 文件选择和解析
  - 导入预览
  - 重复记录检测
  - 自动分类匹配
  - 导入记录管理

#### 2.3.2 未来自动化方向
- 无障碍服务监听支付通知
- 通知栏监听
- 系统级 API 接入

## 3. 非功能需求

### 3.1 性能需求
- 应用启动时间 < 2秒
- 列表滚动流畅（60fps）
- 数据库查询响应 < 100ms

### 3.2 数据安全
- 本地数据加密存储
- 支持数据备份/恢复
- 无云端数据泄露风险

### 3.3 用户体验
- 界面简洁美观（Claude 风格）
- 支持深色/浅色模式
- 操作流畅，动画自然
- 中文界面

### 3.4 兼容性
- Android 5.0 (API 21) 及以上
- 支持主流屏幕尺寸

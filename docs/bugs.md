## 已修复 (Fixed)

### BUG-001: 备注弹窗键盘溢出
**发现时间**: 2026-05-07
**修复时间**: 2026-05-07
**状态**: 已修复
**描述**: 编辑备注时键盘弹出会导致弹窗溢出屏幕
**原因**: AlertDialog 不支持键盘避让
**修复方案**: 改用 showModalBottomSheet + isScrollControlled

### BUG-002: 分类排序无法保存
**发现时间**: 2026-05-07
**修复时间**: 2026-05-07
**状态**: 已修复
**描述**: 分类拖拽排序后刷新列表会恢复原样
**原因**: 分类 Provider 没有刷新触发器
**修复方案**: 添加 categoryRefreshProvider

### BUG-003: ReorderableListView key 错误
**发现时间**: 2026-05-07
**修复时间**: 2026-05-07
**状态**: 已修复
**描述**: 分类管理页面报错 "Every item of ReorderableListView must have a key"
**原因**: `.animate()` 包装后丢失 key
**修复方案**: 将 key 移到外层 Container

### BUG-004: 新增分类删除导致黑屏
**发现时间**: 2026-05-07
**修复时间**: 2026-05-07
**状态**: 已修复
**描述**: 新增的分类在删除时会导致整个应用黑屏
**原因**: 删除分类后关闭编辑对话框时，Navigator.pop 被调用两次（删除确认对话框和编辑对话框），但编辑对话框可能已经被销毁
**修复方案**: 使用 mounted 检查和 try-catch 包裹 Navigator 操作

### BUG-005: 首页预算功能未实现
**发现时间**: 2026-05-07
**修复时间**: 2026-05-07
**状态**: 已修复
**描述**: 首页预算进度卡片只有展示作用，没有实际功能
**原因**: 预算数据为硬编码，未接入数据库
**修复方案**: 使用 budgetUsageProvider 获取真实预算数据，添加跳转到预算管理页面的功能

### BUG-006: 日期筛选无法筛选当天交易
**发现时间**: 2026-05-07
**修复时间**: 2026-05-07
**状态**: 已修复
**描述**: 选择日期范围时，选当天日期无法筛选出当天的交易记录
**原因**: 日期选择器返回的日期时间为 00:00:00，而交易时间有时分秒，导致当天交易被排除
**修复方案**: 在 groupedTransactionsProvider 中修正结束日期为当天 23:59:59

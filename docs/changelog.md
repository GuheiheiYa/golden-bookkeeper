## [1.9.0] - 2026-05-12

### 智能记账 — 支付通知监听自动记账

#### 新增功能
- **Android 通知监听服务**
  - `PaymentNotificationListenerService` 监听微信、支付宝、招商/工商/中国/农业/建设/邮储/平安/中信银行通知
  - `PaymentNotificationParser` 正则解析：金额、商户名、收支方向
  - 去重机制：10秒内相同通知不重复入库
  - SharedPreferences 可配置监听应用白名单

- **双通道数据流**
  - APP 在前台：MethodChannel 实时推送 → 弹出确认弹窗
  - APP 在后台/已杀死：写入 SQLite → 发送系统通知 → 用户点击打开待确认列表

- **前台确认弹窗 (PaymentConfirmSheet)**
  - 显示金额（大字，支出红/收入绿）、商户名、来源 APP
  - 自动匹配分类（关键词匹配）和账户（来源 → 对应账户）
  - 确认 / 忽略操作

- **待确认列表页 (PendingNotificationsScreen)**
  - 列表展示所有 pending 状态记录
  - 支持单条确认/忽略、全部确认、清空所有、下拉刷新

- **通知监听设置页 (NotificationSettingsScreen)**
  - 检测系统通知监听权限状态
  - 未授权引导文案 + 跳转系统设置
  - 已授权时显示各 APP 开关列表

- **个人中心集成**
  - 待确认数量角标显示在"我的"页面入口

#### 改进优化
- 首页快捷操作恢复为：记账/预算/周期/统计
- 首页快捷按钮统一使用 `PageRouteBuilder` + `FadeTransition`
- `RepaintBoundary` 防止 Tab 切换背景重绘闪烁
- `withOpacity` → `withValues(alpha: ...)` 全站统一

#### 技术变更
- 数据库新增 `pending_payments` 表（原生 SQLite，独立于 sqflite）
- `MainActivity` 新增 MethodChannel 桥接（`com.bookkeeper/payment notifications`）
- `lib/app/app.dart` — 生命周期监听

#### 新增文件
- `android/.../PaymentNotificationListenerService.kt` — 通知监听服务
- `android/.../PaymentNotificationParser.kt` — 正则解析器
- `lib/core/services/payment_notification_service.dart` — Flutter 端通知服务
- `lib/features/notification/presentation/notification_settings_screen.dart` — 权限设置页
- `lib/features/notification/presentation/payment_confirm_sheet.dart` — 确认弹窗
- `lib/features/notification/presentation/pending_notifications_screen.dart` — 待确认列表

## [1.8.0] - 2026-05-12

### 个人中心 + 贷款管理 + 还款联动

#### 新增功能
- **"设置"改为"我的"个人中心**
  - 底部导航从"设置"改为"我的"，图标为人像
  - 头像区：金色边框圆形头像 + Lv.X 绿色徽章 + 用户名 + 头衔
  - 等级进度条 + 升级提示
  - 资产摘要卡片：3 张并排卡片（总资产/净资产/负债），从账户余额实时计算
  - 荣誉室：横向滚动成就卡片（魅力之星/预算卫士/储蓄新星/记账达人/分类大师）

- **用户档案系统 (UserProfile)**
  - SharedPreferences 持久化
  - `AssetSummary` provider 从账户表计算总资产/负债/净资产
  - 默认用户：咯噔，Lv.4，精打细算的见习镇长

- **贷款管理**
  - 新增贷款管理页面，支持添加/编辑/删除贷款
  - 贷款类型：房贷 / 车贷 / 信用贷 / 网贷 / 其他
  - 账户管理只显示普通账户（不含贷款），贷款管理只显示贷款

- **分类关联贷款 + 还款自动扣减**
  - 数据库升级到 v3，categories 表新增 `loan_id` 字段
  - 分类管理新增"关联贷款"选择器（仅支出分类显示）
  - 记账时若分类关联了贷款，自动扣减贷款余额
  - 编辑交易时自动调整新旧金额差额
  - 删除交易时自动恢复贷款余额
  - 周期记账产生的还款也自动支持

#### 新增文件
- `lib/features/profile/presentation/profile_provider.dart` — UserProfile + AssetSummary
- `lib/features/profile/presentation/profile_screen.dart` — 个人中心页面
- `lib/features/loan/presentation/loan_list_screen.dart` — 贷款管理页面

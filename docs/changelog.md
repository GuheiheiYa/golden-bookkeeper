# 记账 APP 版本更新日志

## 版本号规则

采用 [语义化版本](https://semver.org/lang/zh-CN/)：
- **主版本号 (MAJOR)**: 不兼容的 API 修改
- **次版本号 (MINOR)**: 向下兼容的功能性新增
- **修订号 (PATCH)**: 向下兼容的问题修正

---

## [1.0.0] - 2026-05-06

### 🎉 首次发布

#### 新增功能
- **核心记账**
  - 收支记录管理（新增、编辑、删除）
  - 分类管理（系统内置 + 自定义）
  - 账户管理（现金、银行卡、支付宝、微信）
  - 交易列表（按日期分组、滑动操作）
  - 自定义数字键盘
  - 分类/账户选择器

- **高级功能**
  - 标签系统（标签 CRUD、标签关联、标签云展示）
  - 多币种支持（币种选择、汇率缓存）
  - 统计报表（饼图、折线图、柱状图、分类排行）
  - 预算管理（总预算、分类预算、超支提醒、环形进度）
  - 周期记账（自动记账规则、后台执行、启用/禁用）
  - 数据导出（CSV、Excel、分享功能）

- **自动化接口**
  - 账单导入接口（微信、支付宝）
  - 重复记录检测
  - 自动分类匹配
  - 导入预览

- **UI/UX**
  - Claude 风格界面（紫色/靛蓝主色调）
  - 深色/浅色/跟随系统模式
  - 流畅动画效果（flutter_animate）
  - 卡片式设计
  - 空状态提示
  - 滑动删除/编辑

- **设置页面**
  - 主题切换
  - 账户管理入口
  - 分类管理入口
  - 标签管理入口
  - 预算管理入口
  - 周期记账入口
  - 数据导入/导出

#### 技术实现
- Flutter 跨平台框架
- Riverpod 状态管理
- sqflite SQLite 数据库
- go_router 路由管理
- fl_chart 图表库
- flutter_animate 动画库

#### 数据库
- 8 张数据表
- 完整的 CRUD 操作
- 默认数据初始化

## [1.1.0] - 2026-05-07 (上午)

### 🚀 功能完善

#### 新增功能
- **记账页面增强**
  - 标签选择器（支持多选标签，带颜色标识）
  - 交易编辑模式（通过路由 `/transaction/edit/:id`）
  - 币种选择器（支持 CNY/USD/EUR/GBP/JPY/KRW/HKD/TWD）

- **交易明细页增强**
  - 搜索功能（按备注和分类名搜索）
  - 筛选功能（收支类型、日期范围、分类多选、账户筛选）
  - 筛选条件标签栏（显示活跃筛选条件，支持单个清除和全部清除）
  - 左滑编辑、右滑删除
  - 交易详情弹窗（显示完整信息，支持编辑和删除操作）

- **周期记账自动执行**
  - 进入首页时自动检查并执行到期的周期记账规则
  - 执行后显示浮动提示消息（包含执行条数和"查看"按钮）

- **多币种支持**
  - 记账时可选择币种（8种常用货币）
  - 汇率缓存到本地数据库
  - 币种选择器（Claude 风格网格布局）

- **删除功能完善**
  - 分类删除前显示关联交易数量警告
  - 账户删除前显示关联交易数量警告
  - 所有列表都有删除确认对话框

#### 改进优化
- 主题模式切换样式优化（SegmentedButton 全宽布局）
- AppCard 组件边距优化

#### 技术变更
- 使用 sqflite 替代 drift 作为数据库方案
- 添加 universal_io 包支持 Web 平台
- 升级 file_picker 到 ^8.0.0

## [1.2.0] - 2026-05-07 (下午)

### 🚀 UI/UX 优化

#### 新增功能
- **计算器功能**
  - 数字键盘 +/- 按钮实现计算器逻辑
  - 按 +/- 后在金额上方显示计算表达式
  - 支持连续计算

  - 图标数量从 20+ 扩展到 90+
  - 图标按分类分组（餐饮、交通、购物、娱乐等 12 类）
  - 图标选择器显示分类标题和 Tooltip

- **日期选择器中文化**
  - 添加 flutter_localizations 支持
  - 日期选择器按钮和提示显示中文

- **文档完善**
  - 创建 README.md 项目介绍
  - 创建 docs/changes.md 变更记录文档
  - 更新 SKILL.md 强制文档更新规则

#### 问题修复
- 修复分类管理 ReorderableListView key 错误
- 修复分类排序无法保存的问题
- 修复备注弹窗键盘溢出问题
- 修复提示框不自动消失的问题
- 修复首页 FAB 遮挡交易金额问题
- 修复分类对话框保存按钮需要滚动才能看到

#### 改进优化
- 数字键盘布局优化（00 改为日期选择按钮）

## [1.2.1] - 2026-05-07 (晚上)

### 🚀 功能优化

#### 新增功能
- **首页预算功能**
  - 预算进度卡片接入真实数据库数据

- **交易明细页添加按钮**
  - AppBar 添加记账入口
  - 方便从明细页快速添加交易

#### 问题修复
- 修复日期筛选无法筛选当天交易的问题
- 修复筛选面板"应用筛选"按钮需要滚动才能看到的问题
- 修复分类删除导致黑屏的问题

- 筛选面板布局优化（按钮固定在底部）
- 更新 update-docs skill 描述，优化自动触发机制

## 7. 账单导入

### 支持来源
| 来源 | 格式 | 编码 |
|------|------|------|
| 微信支付 | CSV / XLSX | UTF-8 |
| 支付宝 | CSV / XLSX | GBK |

### 导入流程
1. 选择账单文件 → 2. 解析格式 → 3. 预览 → 4. 确认导入 → 5. 重复检测 → 6. 自动分类匹配 → 7. 保存

### 自动分类匹配策略
1. **交易类型直接映射**: 转账→转账、群收款→其他、滴滴出行→交通
2. **描述关键词匹配**: 根据交易对方名称匹配（美团→餐饮、滴滴→交通）
3. **来源分类字段**: 支付宝的"餐饮美食"等有意义分类名
4. **兜底**: 固定使用"其他"分类

### 重复检测
基于金额 + 交易时间（±1分钟容差） + 交易对方 + 来源组合判断

## 10. 智能记账（v1.9.0）

### 通知监听服务
- [x] Android NotificationListenerService 监听系统通知
- [x] 支持微信、支付宝、招商/工商/中国/农业/建设/邮储/平安/中信/民生/厦门银行
- [x] 正则解析通知文本：提取金额、商户名、收支方向
- [x] 去重机制：5秒内相同金额的通知不重复入库
- [x] SharedPreferences 可配置监听应用白名单（仅存储移除列表，新增银行自动生效）

### 数据流
检测到支付 → 静默写入 SQLite `pending_payments` 表（去重 5 秒）→ 用户打开待确认列表处理

### 待确认列表
- [x] 单条确认（自动匹配"其他"分类 + 默认账户）
- [x] 忽略（标记为已处理）
- [x] 全部确认（批量自动匹配）
- [x] 清空所有
- [x] 下拉刷新
- [x] APP 恢复前台时自动刷新

### 权限设置
- [x] 检测通知监听权限状态，未授权引导跳转系统设置
- [x] 已授权时显示各 APP 开关列表

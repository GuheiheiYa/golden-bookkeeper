## 1. 核心记账模块

### 1.1 交易记录管理

#### 功能列表
- [x] 新增交易记录
- [x] 编辑交易记录（支持从列表左滑、长按快捷操作或详情弹窗进入）
- [x] 删除交易记录（支持右滑删除、长按快捷操作，带确认对话框）
- [x] 查看交易详情（底部弹窗显示完整信息）

- [x] 按日期分组显示
- [x] 按分类/标签/账户筛选
- [x] 关键字搜索（按备注和分类名）
- [x] 多条件组合筛选（收支类型、日期范围、分类、账户）

#### 数据模型
```dart
class Transaction {
  final int id;
  final double amount;
  final String currency;
  final double exchangeRate;

  final String? note;
  final DateTime date;
  final int categoryId;
  final int accountId;
  final int? recurringRuleId;
  final String? imagePath;
  final DateTime createdAt;
  final DateTime? updatedAt;
}
```

#### 界面说明
- **记账页面**: 顶部 Tab 切换收入/支出，底部数字键盘，分类选择网格
- **交易列表**: 按日期分组，显示每日小计，支持滑动删除/编辑
- **交易详情**: 显示完整交易信息，支持编辑和删除

## 7. 账单导入

### 7.1 支持来源

#### 微信支付
- 文件格式: CSV (UTF-8)
- 关键字段:
  - 交易时间
  - 交易类型
  - 交易对方
  - 商品
  - 收/支
  - 金额
  - 支付方式

#### 支付宝
- 文件格式: CSV (GBK 编码)
- 关键字段:
  - 交易时间
  - 交易分类
  - 交易对方
  - 商品说明
  - 收/支
  - 金额
  - 收/付款方式

### 7.2 导入流程

#### 步骤
1. 选择账单文件
2. 解析文件格式
3. 预览导入记录
4. 确认导入
5. 重复检测
6. 自动分类匹配
7. 保存到数据库

#### 功能列表
- [x] 文件选择和解析
- [x] 导入预览
- [x] 重复记录检测
- [x] 自动分类匹配
- [x] 导入记录管理
- [x] 导入进度显示

### 7.3 重复检测策略

基于以下字段组合判断重复:
- 金额 (amount)
- 交易时间 (date) ± 1 分钟容差
- 交易对方 (description)
- 来源 (source: wechat/alipay)
